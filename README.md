
# Objetivo General

## Descripción del Proyecto

Actualmente la empresa constructora administra sus obras mediante WhatsApp, llamadas telefónicas y planillas en papel, lo que dificulta el seguimiento de los proyectos, la asignación de empleados y el control del avance de las tareas.

El proyecto propone desarrollar una aplicación web que centralice toda la información de la empresa y permita optimizar la gestión de las obras.

## Objetivo SMART

Desarrollar e implementar, en un plazo de **tres meses**, una aplicación web que permita:

- Registrar obras.
- Registrar empleados.
- Asignar empleados a obras.
- Registrar y asignar tareas.
- Controlar el estado de las tareas.
- Consultar el avance de las obras.

Con el objetivo de mejorar la organización, reducir errores administrativos y facilitar el seguimiento de los proyectos de construcción.

## Beneficios esperados

- Centralización de la información.
- Mejor organización del personal.
- Seguimiento del avance de las obras.
- Reducción del uso de planillas y registros manuales.
- Mayor rapidez en la consulta de información.
